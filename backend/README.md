# Backend

Contains the backend logic for the SAI system, including API endpoints, database connections, and business logic.

## Structure

- `routes/`: API routes.
- `controllers/`: Business logic.
- `models/`: Database models.
- `middlewares/`: Custom middleware.

## Setup

Run `npm install` to install dependencies. Use `npm run dev` to start the development server.

## Notes

Ensure the `.env` file is configured for the database connection.
